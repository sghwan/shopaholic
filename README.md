## 🏬SHOPAHOLIC
SHOPAHOLIC은 온라인 사용자들을 위한 기업 과제 형식의 웹 쇼핑몰 서비스 개발 프로젝트 입니다.

## 📰 서비스 구현 의도 
1. 여러 상품을 카테고리별로 손쉽게 쇼핑을 즐기도록 구성했습니다.
2. 인기 있는 상품을 볼 수 있도록 리뷰와 평점 기능을 구현했습니다.
